All public symbols are available from the package root:

```python
from neograph import (
    Node, Tool, Construct, Oracle, Each, Operator,
    compile, run, configure_llm, raw_node,
    register_scripted, register_condition, register_tool_factory,
)
```

## Node

A typed processing block that compiles to a LangGraph node.

```python
class Node(BaseModel):
    name: str
    mode: Literal["produce", "gather", "execute", "scripted"] = "produce"
    input: type[BaseModel] | None = None
    output: type[BaseModel] | None = None
    model: str | None = None
    prompt: str | None = None
    llm_config: dict[str, Any] = {}
    tools: list[Tool] = []
```

**Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `name` | `str` | Unique node name. Becomes the state field name (hyphens become underscores). |
| `mode` | `"produce" \| "gather" \| "execute" \| "scripted"` | Execution mode. `produce`: single LLM call. `gather`: ReAct tool loop (read-only). `execute`: ReAct tool loop (mutations). `scripted`: pure Python. |
| `input` | `type[BaseModel] \| None` | Pydantic model consumed from state. `None` for the first node in a pipeline. |
| `output` | `type[BaseModel] \| None` | Pydantic model written to state. Required for all nodes. |
| `model` | `str \| None` | LLM tier name passed to `llm_factory`. E.g., `"fast"`, `"reason"`. |
| `prompt` | `str \| None` | Prompt template name passed to `prompt_compiler`. |
| `llm_config` | `dict[str, Any]` | LLM settings forwarded to `llm_factory`. Common keys: `temperature`, `max_tokens`, `output_strategy` (`"structured"`, `"json_mode"`, `"text"`). |
| `tools` | `list[Tool]` | Tools available to gather/execute mode nodes. |

**Pipe operator:**

```python
node | Oracle(n=3, merge_fn="merge")
node | Each(over="clusters.groups", key="label")
node | Operator(when="needs_review")
node | Oracle(n=3, merge_fn="m") | Operator(when="check")
```

**Example:**

```python
decompose = Node(
    name="decompose",
    mode="produce",
    output=Claims,
    model="fast",
    prompt="req/decompose",
    llm_config={"temperature": 0.7},
)
```

## Node.scripted()

Class method that creates a deterministic node with no LLM.

```python
@classmethod
def scripted(
    cls,
    name: str,
    fn: str,
    input: type[BaseModel] | None = None,
    output: type[BaseModel] | None = None,
) -> Node
```

**Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `name` | `str` | Unique node name. |
| `fn` | `str` | Name of a function registered with `register_scripted()`. |
| `input` | `type[BaseModel] \| None` | Pydantic model consumed from state. |
| `output` | `type[BaseModel] \| None` | Pydantic model written to state. |

**Example:**

```python
register_scripted("classify", classify_fn)
classify = Node.scripted("classify", fn="classify", input=Claims, output=ClassifiedClaims)
```

## Tool

An LLM-callable tool with a per-tool call budget.

```python
class Tool(BaseModel, frozen=True):
    name: str
    budget: int = 0
    config: dict[str, Any] = {}
```

**Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `name` | `str` | Tool name. Must match a factory registered with `register_tool_factory()`. |
| `budget` | `int` | Maximum calls allowed. `0` means unlimited. |
| `config` | `dict[str, Any]` | Per-tool configuration passed to the tool factory. |

**Example:**

```python
tools = [
    Tool(name="search_codebase", budget=5),
    Tool(name="read_artifact", budget=10, config={"max_chars": 6000}),
]
```

## Construct

An ordered composition of Nodes and sub-Constructs that compiles to a LangGraph `StateGraph`.

```python
class Construct(BaseModel):
    name: str
    description: str = ""
    nodes: list[Node | Construct] = []
    input: type[BaseModel] | None = None
    output: type[BaseModel] | None = None
```

**Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `name` | `str` | Pipeline name (positional or keyword). |
| `description` | `str` | Optional human-readable description. |
| `nodes` | `list[Node \| Construct]` | Ordered list of nodes and sub-constructs. Execution order follows list order. |
| `input` | `type[BaseModel] \| None` | Required when used as a sub-construct. Declares input boundary. |
| `output` | `type[BaseModel] \| None` | Required when used as a sub-construct. Declares output boundary. |

**Top-level pipeline:**

```python
pipeline = Construct("my-pipeline", nodes=[node_a, node_b, node_c])
```

**Sub-construct (subgraph with isolated state):**

```python
enrich = Construct(
    "enrich",
    input=Claims,
    output=ScoredClaims,
    nodes=[lookup, score],
)
parent = Construct("parent", nodes=[decompose, enrich, report])
```

**With modifiers:**

```python
enrich | Oracle(n=3, merge_fn="merge")
enrich | Each(over="items.groups", key="label")
```

## Oracle

Modifier: N-way ensemble fan-out + merge. Exactly one of `merge_prompt` or `merge_fn` must be provided.

```python
class Oracle(Modifier, frozen=True):
    n: int = 3
    merge_prompt: str | None = None
    merge_model: str = "reason"
    merge_fn: str | None = None
```

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `n` | `int` | `3` | Number of parallel generators. |
| `merge_prompt` | `str \| None` | `None` | Prompt template name for LLM judge merge. |
| `merge_model` | `str` | `"reason"` | LLM tier for merge (when using `merge_prompt`). |
| `merge_fn` | `str \| None` | `None` | Registered scripted merge function name. |

**Example:**

```python
# Scripted merge
node | Oracle(n=3, merge_fn="deduplicate_claims")

# LLM merge
node | Oracle(n=5, merge_prompt="rw/merge-judge", merge_model="fast")
```

## Each

Modifier: fan-out over a collection, process each item in parallel.

```python
class Each(Modifier, frozen=True):
    over: str
    key: str
```

**Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `over` | `str` | Dotted path to the collection in state. E.g., `"discover.groups"` resolves to `state.discover.groups`. |
| `key` | `str` | Field on each collection item used as the result dictionary key. |

Results are collected as `dict[str, output_type]`.

**Example:**

```python
verify = Node.scripted(
    "verify", fn="verify_cluster", input=ClusterGroup, output=VerifyResult
) | Each(over="discover.groups", key="label")

# result["verify"] -> {"auth": VerifyResult(...), "logging": VerifyResult(...)}
```

## Operator

Modifier: human-in-the-loop interrupt. Requires a checkpointer at `compile()` time.

```python
class Operator(Modifier, frozen=True):
    when: str
```

**Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `when` | `str` | Name of a condition function registered with `register_condition()`. |

The condition function receives the full state. Return a truthy value to interrupt (the value becomes the interrupt payload). Return `None` to continue.

**Example:**

```python
register_condition("needs_review", lambda state: (
    {"issues": state.check.issues}
    if state.check and not state.check.passed
    else None
))

check = Node.scripted("check", fn="validate", ...) | Operator(when="needs_review")
graph = compile(pipeline, checkpointer=MemorySaver())
```

## compile()

Compile a Construct into an executable LangGraph `StateGraph`.

```python
def compile(
    construct: Construct,
    checkpointer: Any = None,
) -> CompiledStateGraph
```

**Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `construct` | `Construct` | The pipeline to compile. |
| `checkpointer` | `Any \| None` | LangGraph checkpointer instance. Required if any node uses `Operator`. |

**Returns:** A compiled LangGraph `StateGraph`.

**Raises:** `ValueError` if `Operator` is used without a checkpointer.

**Example:**

```python
graph = compile(pipeline)
graph = compile(pipeline, checkpointer=MemorySaver())
```

## run()

Execute a compiled NeoGraph graph.

```python
def run(
    graph: CompiledStateGraph,
    input: dict[str, Any] | None = None,
    resume: dict[str, Any] | None = None,
    config: dict[str, Any] | None = None,
) -> dict[str, Any]
```

**Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `graph` | compiled graph | Output of `compile()`. |
| `input` | `dict \| None` | Initial state values. All fields are also injected into `config["configurable"]`. |
| `resume` | `dict \| None` | Human feedback for resuming after an `Operator` interrupt. |
| `config` | `dict \| None` | LangGraph `RunnableConfig`. Must include `thread_id` for interrupt/resume. |

Either `input` or `resume` must be provided, not both.

**Returns:** A dict of node results. Keys are node names (hyphens become underscores). Values are typed Pydantic model instances. Framework internal fields (`neo_*`) are stripped. If the graph was interrupted, the result includes an `"__interrupt__"` key.

**Example:**

```python
# First run
result = run(graph, input={"node_id": "REQ-001"})

# With config for checkpointing
config = {"configurable": {"thread_id": "review-001"}}
result = run(graph, input={"node_id": "REQ-001"}, config=config)

# Resume after interrupt
result = run(graph, resume={"approved": True}, config=config)
```

## configure_llm()

Configure the LLM layer. Must be called before using any LLM node (`produce`, `gather`, `execute`).

```python
def configure_llm(
    llm_factory: Callable,
    prompt_compiler: Callable,
) -> None
```

**Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `llm_factory` | `Callable` | Creates LLM instances. Simple: `(tier) -> BaseChatModel`. Advanced: `(tier, node_name=, llm_config=) -> BaseChatModel`. |
| `prompt_compiler` | `Callable` | Builds message lists. Simple: `(template, input_data) -> list[BaseMessage]`. Advanced: `(template, input_data, node_name=, config=) -> list[BaseMessage]`. |

Both signatures are auto-detected via `TypeError` fallback -- simple and advanced forms work without configuration.

**Example:**

```python
# Simple
configure_llm(
    llm_factory=lambda tier: ChatOpenAI(model=MODELS[tier]),
    prompt_compiler=lambda template, data: [HumanMessage(content=str(data))],
)

# Production
def my_factory(tier, node_name=None, llm_config=None):
    return ChatOpenAI(
        model=MODELS[tier],
        temperature=(llm_config or {}).get("temperature", 0),
    )

def my_compiler(template, data, *, node_name=None, config=None):
    node_id = config["configurable"]["node_id"]
    return build_prompt(template, node_id, data)

configure_llm(llm_factory=my_factory, prompt_compiler=my_compiler)
```

## register_scripted()

Register a deterministic function for use with `Node.scripted()` and `Oracle(merge_fn=...)`.

```python
def register_scripted(name: str, fn: Callable) -> None
```

**Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `name` | `str` | Lookup name. Must match `fn=` on `Node.scripted()` or `merge_fn=` on `Oracle`. |
| `fn` | `Callable` | For nodes: `(input_data, config) -> BaseModel`. For Oracle merge: `(variants: list, config) -> BaseModel`. |

**Example:**

```python
def extract_text(input_data, config):
    return RawText(text="extracted content")

register_scripted("extract_text", extract_text)
Node.scripted("extract", fn="extract_text", output=RawText)
```

## register_condition()

Register a condition function for use with `Operator(when=...)`.

```python
def register_condition(name: str, fn: Callable) -> None
```

**Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `name` | `str` | Lookup name. Must match `when=` on `Operator`. |
| `fn` | `Callable` | `(state) -> dict \| None`. Return a truthy dict to trigger interrupt (the dict becomes the payload). Return `None` to continue. |

**Example:**

```python
def needs_review(state):
    val = state.check
    if val and not val.passed:
        return {"issues": val.issues, "message": "Review required"}
    return None

register_condition("needs_review", needs_review)
```

## register_tool_factory()

Register a factory that creates tool instances for `gather`/`execute` mode nodes.

```python
def register_tool_factory(name: str, fn: Callable) -> None
```

**Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `name` | `str` | Tool name. Must match `Tool(name=...)`. |
| `fn` | `Callable` | `(config: dict, tool_config: dict) -> tool_instance`. The tool must have a `.name` attribute and an `.invoke(args)` method. |

**Example:**

```python
register_tool_factory(
    "search_codebase",
    lambda config, tool_config: SearchTool(
        api_key=config["configurable"]["api_key"],
        max_results=tool_config.get("max_results", 10),
    ),
)
```

## raw_node

Decorator that wraps a classic LangGraph function as a NeoGraph `Node`. The framework wires edges, observability, and state around it; the function body is yours.

```python
@raw_node(input=SomeInput, output=SomeOutput)
def custom_logic(state, config):
    ...
```

**Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `input` | `type[BaseModel] \| None` | Pydantic model consumed from state. |
| `output` | `type[BaseModel] \| None` | Pydantic model written to state. |

**Returns:** A `Node` instance. The function name (underscores replaced by hyphens) becomes the node name.

**Example:**

```python
@raw_node(input=ConsolidatedResult, output=ConsolidatedResult)
def custom_resolution(state, config):
    result = state.consolidated
    # ... custom logic with full state access ...
    return {"custom_resolution": modified_result}

pipeline = Construct("my-pipeline", nodes=[other_node, custom_resolution])
```
